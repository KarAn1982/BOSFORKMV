# БОСФОР — защита загружаемых ZIP

Дата: 20 июня 2026 года

Prototype API теперь анализирует ZIP central directory до помещения лида в
очередь. Блокируются:

- более 500 файлов;
- суммарный распакованный объём более 200 МБ;
- compression ratio более 100:1;
- зашифрованные записи;
- абсолютные пути и `../` path traversal;
- исполняемые и скриптовые форматы внутри архива;
- повреждённый ZIP без central directory.

Проверка не распаковывает пользовательский архив на диск. Production-цепочка
дополнительно требует sandboxed extraction, recursive archive limits и
антивирус после загрузки в закрытое quarantine-хранилище.
