# БОСФОР — карта контента Alutech

Источник: папка `Для сайтов партнеров` и книга `Сводная.xlsx`.

## Доступные группы

| Группа | Системы |
|---|---|
| Тёплые и холодные окна | ALT W62, W72, W77, C43, C48, скрытые створки, вентстворки, алюмодеревянные решения |
| Двери | ALT W62/W72, террасные, крупногабаритные, структурные, C43/C48 |
| Порталы | ALT SL130, SL160, BF73 |
| Фасады | ALT F50, F60, F50 SG/SSG/HC/FW, SKL50, USKL60, SP50 |
| Навесные вентилируемые фасады | ALT 150 |
| Противопожарные решения | ALT FR76, ALT W72 FR |
| Балконы и вентиляционные решётки | ALT 100P, ALT VR26 |
| Интерьерные решения | ALT 110, 111, 115, 118 |
| Ручки и фурнитура | ROTO, HAUTAU, DUBLIN, STUBLINA, ALUTECH, MEDOS, HOPPE, DEBAR |

## Использование на сайте

- `alutech.html` — общая матрица направлений и маршрутизация;
- `aluminum-windows.html` — W62/W72/W77, C43/C48;
- `doors.html` — W62/W72, C43/C48, террасные и структурные решения;
- `portal-systems.html` — SL130, SL160, BF73;
- `facades.html` — F50/F60, SG/SSG/HC/FW, купольные и солнцезащитные решения;
- будущие страницы — НВФ, противопожарные и интерьерные системы только после
  подтверждения спроса и технической редакции.

## Ограничения

Наличие файлов производителя не доказывает официальный партнёрский статус.
Технические показатели, классы, размеры и сертификаты не публикуются до проверки
конкретного документа и срока его действия.
