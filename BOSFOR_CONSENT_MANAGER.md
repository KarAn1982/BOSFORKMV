# БОСФОР — consent manager

Дата: 20 июня 2026 года

`website/consent-manager.js` реализует prototype CMP:

- аналитика по умолчанию запрещена;
- два равнозначных решения: только необходимые / разрешить аналитику;
- выбор сохраняется с версией и датой;
- настройки можно открыть повторно;
- `Escape` выбирает только необходимые;
- analytics loader получает событие только после согласия;
- баннер доступен с клавиатуры и имеет `role="dialog"`.

Текст intentionally нейтральный и ссылается на draft privacy page. Production
публикация требует юридического решения по cookies/analytics, реквизитов
оператора, сроков хранения и фактического списка обработчиков.

Изменение версии consent требует нового storage key или migration, чтобы
пользователь повторно принял существенно изменённые условия.
