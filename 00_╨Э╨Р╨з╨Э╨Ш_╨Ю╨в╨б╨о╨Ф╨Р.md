# БОСФОР — пакет для запуска сайта в WordPress

Порядок для администратора (жены):

1. **`ШАБЛОНЫ/`** — 6 файлов-шаблонов (по типу страниц). Сделать 6 шаблонов в WP по ним. Памятка внутри: `0_ЧИТАЙ_МЕНЯ.md`.
2. **`prototypes/`** + **`_КАРТА_ПРОТОТИПОВ.html`** — все 45 готовых страниц как контент-эталон (открываются в браузере).
3. **`визуалы-страниц/`** — как каждая страница выглядит (десктоп + мобайл), 93 скриншота. Удобно сверять вид.
4. **`СТРУКТУРА_И_ШАБЛОНЫ.md`** — секции внутри шаблонов и переиспользуемые блоки.
5. **SEO/GEO:** `SEO_GEO_ДЛЯ_WORDPRESS.md` + `SEO_META_ПО_СТРАНИЦАМ.csv` + `seo-geo-файлы/` (robots, llms.txt, карты сайта, ключ IndexNow в корень).
6. **`редиректы/`** — `РЕДИРЕКТЫ_84.csv` для плагина Redirection (сохранить поисковый вес старого сайта и B2B-домена).
7. **`форма-заявки/`** — поля и маршрутизация формы заявки (главный KPI — лиды).

Отдельно для владельца/CMO:

- **`ВЛАДЕЛЬЦУ_подтвердить_факты.md`** — что подтвердить ДО запуска (в т.ч. конфликт телефона). Без этого не синхронизировать карточки Яндекс/Google Бизнес.

Справочно (необязательно к действию): **`материалы-релиза/`** — миграционный runbook, CMS-модели, тех-архитектура, реестр фактов, launch-гейты от Codex.

> Серверная часть релиза (Node-сервер, nginx, edge-кэш) в пакет НЕ включена — она для статического хостинга, в WordPress не используется.
